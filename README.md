# 🌱 Plant Disease Detection Application

An AI-powered web application that detects plant diseases from leaf images using Computer Vision and Deep Learning techniques.

## 📖 Overview

Plant diseases significantly impact agricultural productivity and crop quality. This project provides an intelligent solution for automatic plant disease detection by analyzing leaf images using a trained MobileNet deep learning model.

The application enables users to upload plant leaf images and receive instant disease predictions, helping farmers and agricultural experts make informed decisions.

---

## 🚀 Features

- Upload plant leaf images
- Automatic disease detection
- Deep Learning-based classification
- Fast and accurate predictions
- User-friendly web interface
- REST API integration
- Scalable architecture

---

## 🏗️ System Architecture

The system follows a three-tier architecture:

### Frontend Layer
- React.js
- Vite
- HTML5
- CSS3
- JavaScript

### Backend Layer
- Node.js
- Express.js

### Machine Learning Layer
- Flask
- TensorFlow
- Keras
- MobileNet

---

## 🔄 Workflow

1. User uploads a plant leaf image.
2. React frontend sends the image to the Node.js backend.
3. Backend forwards the image to the Flask ML service.
4. MobileNet model processes the image.
5. Disease prediction is generated.
6. Result is returned and displayed to the user.
